## 🚀 Como rodar localmente

1. Certifique-se de ter o Docker instalado.
2. Execute:
   ```bash
   docker-compose up
   ```
3. Acesse http://localhost:8000.

## 📂 Estrutura
- `docs/`: Arquivos de documentação (Markdown).
- `mkdocs.yml`: Configuração do site.
