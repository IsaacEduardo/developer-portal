# Portal da Plataforma de Desenvolvimento

## 1. Introdução e Contexto
Com vários templates, pipelines e padrões, é comum perder-se: “onde está o template X?”, “como se usa o pipeline Y?”.

Este **Developer Portal** serve como ponto central para:
- Documentação dos standards (templates, pipelines, boas práticas).
- Catálogo de serviços e projetos ativos.
- Golden Paths para acelerar o desenvolvimento.

## 2. Objetivo
Facilitar a vida dos desenvolvedores fornecendo uma "homepage" para a IDP (Internal Developer Platform) acadêmica.

## 3. Acesso Rápido
- **[Golden Paths](golden-paths/api-projeto-1.md)**: Guias passo-a-passo (APIs, Microserviços, Pipelines).
- **[Catálogo de Serviços](catalogo.md)**: Lista de repositórios e status dos projetos.
